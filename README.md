# PlayerControl for BIGTET
- 「PlayerControl for BIGTET」は、対戦ゲームのプレイヤー名などを表示する「StreamControl」互換のアプリケーションです。
- TETRIS the Grandmaster（TGM1）ビッグモードのフリープレイ・各種対戦用に合わせて設計しています。
